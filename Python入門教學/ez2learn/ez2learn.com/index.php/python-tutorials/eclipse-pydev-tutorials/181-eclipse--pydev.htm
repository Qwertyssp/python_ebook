安裝Eclipse 與 Pydev
作者是 Victor   
週三, 11 二月 2009 21:09

下載並安裝Eclipse

Eclipse可以在它的官方網站Eclipse.org找到並下載，通常我們選擇Eclipse Classic版本，下載完成後是一個壓縮檔，可能令人覺得奇怪為何沒有安裝檔，事實上只要將壓縮檔案內的eclipse資料夾，解壓縮到你想安裝的目錄中就算安裝完成了

常見問題

當你執行安裝完成的eclipse資料夾裡的eclipse.exe，如果出現下面錯誤訊息

A Java Runtime Environment or Java Development Kit must be available in order to run Eclipse. No Java Virtual Machine was found after searching the following locations

c:\eclipse\jre\bin\javaw.exe
javaw.exe in your current PATH  

這表示你沒有安裝Java的執行環境，要執行eclipse必須安裝JRE或JDK，你可以到

Java官方網站的JRE下載頁面

找到JRE下載並安裝

安裝PyDev

當你安裝完eclipse並且可以執行後，就可以安裝PyDev，執行eclipse它會顯示一個視窗

要你選擇工作用的資料夾，Eclipse的設定還有專案都會放進此資料夾，你可以改成你想要的資料夾，或是直接按OK使用它預設的資料夾

當Eclipse開啟後，從選單按Help -> Software Updates，如下圖所示

然後選擇Available Software頁面，接著按Add site，貼上下列網址

http://pydev.sourceforge.net/updates/

如圖所示

完成後按Ok，接著點擊 PyDev的"+"，展開PyDev的節點，接著要等一小段時間，讓它上網抓取PyDev相關的套件，當完成後會多出PyDev的相關套件在子節點裡，勾選它們然後按Install

接著Eclipse就會開始下載PyDev的套件，需要一小段時間，接著出現如下圖所示的視窗

接著按Next進行下個步驟，它會要求你接受條款，選擇I accept the terms of the license agreements然後按Finish

接著它會要你重新啟動Eclipse，按Yes就算安裝完成了

設定PyDev

雖然安裝完成了，但是還不能直接使用，我們還需要設定PyDev，如下圖所示，我們選擇Window -> Preferences來設定PyDev

接著開啟設定視窗，我們首先設定檔案的編碼為UTF8，這麼做是為了讓檔案使用UTF8編碼，如果程式中有中文比較不易出問題

接著是設定Python直譯器的路徑，從Pydev的Interpreter - Python頁面選擇New

之後會跳出一個視窗讓你選擇Python直譯器的檔案位置，選擇你安裝的Python直譯器的所在位置

按開啟後會執行一些動作來搜尋相關的函式庫，接著按OK，同樣會執行一些動作，完成之後PyDev就設定完成，可以開始使用
