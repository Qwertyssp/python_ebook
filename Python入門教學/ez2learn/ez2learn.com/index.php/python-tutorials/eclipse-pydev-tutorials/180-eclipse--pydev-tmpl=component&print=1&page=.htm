什麼是Eclipse 與 Pydev?

Eclipse是一款用Java寫成的開放式的IDE平台，有活躍的社群和許多的外掛、插件，而Pydev就是其中一款用來擴充Eclipse的插件，安裝了之後就可以使用Eclipse來開發Python，或許有人覺得我們已經有了Python內建的介面，為什麼還要有Eclipse和Pydev來開發Python? 原因在於官方的介面太過陽春，開發大型程式相當不方便，因此我們需要功能更完備的Python IDE，Eclipse的Pydev就是相當不錯的選擇，目前我所有的Python專案都是使用Pydev開發，以下是使用Pydev開發中的專案畫面

接著介紹它的特點

Eclipse 與 Pydev的特色

專案管理

對於大型程式來說，有專案管理的介面是很重要的，所有檔案都可以一目瞭然

強大的除錯器支援

它提供良好的除錯器介面，讓除錯工作相當輕鬆

強大的自動完成功能

基本上只要是能夠由程式碼直接推斷出形別的物件，幾乎都有自動完成的功能，而各種括號的輸入，一樣有自動完成的功能，這點可以說是它最強的地方

優良的介面

例如Traceback印出來時，自動將程式碼錄徑標成連結，點擊就可以直接跳到該檔案的該行程式碼，使用上相當方便

需要相當多記憶體

是的，Eclipse會吃掉不少的記憶體，很多同樣是用Java開發出來的東西都有這樣的毛病，如果你在記憶體很少的環境執行會相當痛苦
