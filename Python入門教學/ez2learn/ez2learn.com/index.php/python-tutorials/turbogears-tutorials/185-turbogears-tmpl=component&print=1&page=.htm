設定Python路徑

如果你已經安裝完Python，接著要安裝TurboGears就非常的簡單，首先要做的是設定Python的路徑，在"我的電腦"上點右鍵，選擇"內容"，找到"進階"的選項，切換到該頁面後，往下找有一個"環境變數"。
進到環境變數之後，在下面的"系統變數"的欄位裡，找Path這一欄，選擇後按編輯。
接著在"變數值"的欄位，將游標移到最後面，貼上後按確定

;C:\Python25;C:\Python25\Scripts

請注意不要改到原本前面有的其它值，可能會造成其它程式出問題，找不到程式檔等
如果你的Python安裝在其它地方，請修改前面的C:\Python25的值成你安裝Python的位置
當完成以上步驟後，安裝TurboGears最麻煩的步驟都已經完成了。

安裝TurboGears

安裝TurboGears極度地簡單，首先到TurboGears的官方網站下載TurboGears的自動安裝檔


http://docs.turbogears.org/1.0/Install


找到 tgsetup.py 這個檔案後下載


http://www.turbogears.org/download/tgsetup.py


接著打開Windows的命令提示字元，切換到放tgsetup.py這個檔案的目錄，然後打下面這行後按enter

python tgsetup.py

或是直接點該檔案兩下，它就會開始執行。
需要一小段時間自動上網去抓最新的版本並且安裝，跑完就是安裝完成。
