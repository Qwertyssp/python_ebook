什麼是TurboGears

TurboGears是一套以Python寫的網頁程式開發用的快速網頁開發框架

官方的介紹

我們引用部份TurboGears官方網站的介紹來說明TurboGears的特點。

專案目的

快速地建立極佳的網頁應用程式，這就是這個框架的主要目的，不止快速，而且讓它更容易、更有趣。

專案哲學

TurboGears整合了各種非常好的Open source專案，而不是重新打造它們。

執照

TurboGears釋放於liberal open source licenses，這讓使用者可以自由使用TurboGears在商業、和非商業用途

著名的20分鐘寫一個Wiki影片

百聞不如一見，TurboGears有一段著名的20分鐘寫一個Wiki影片，展示TurboGears開發到底有多快速。

為什麼選擇TurboGears?

優秀的組件

它組合了各種優秀的組件，每個組件都是Open source的專案，這代表著它們都相當的完備

可替換的組件

整合不同的組件，它的並不是將組件寫死，相反地它給你選擇，如果你不喜歡預設的某個組件，你可以換掉它

高度的彈性

從頭到尾的Open source專案組合，這表示你可以修改任何部份，低藕合的各種組件，也帶來極大的彈性

良好的安全性

使用Python寫成，天生就沒有buffer overflow的問題，而使用ORM存取資料庫，同樣也免去SQL Injection的常見漏洞。

省去常見的工作

與生俱來的很多特點讓使用者不用做重覆的工作，例如授權系統，你不必替它寫一套使用者帳號密碼驗證的程式，它本身就俱有這樣的功能

使用ORM存取資料庫

使用ORM存取資料庫除了免去SQL Injection的問題，而且只要寫一次，就可以使用在不同的資料庫上，而且你不必寫半行的SQL語法，目前預設的ORM是SQLObject，未來會改成SQLAlchemy。

MVC架構

使用MVC架構讓藕合降低，表示彈性增加

設計師有善的樣板系統

使用Kid做為預設的樣板系統，語法是使用XML的語法，設計師可以在瀏覽器開啟事先預覽頁面的樣子，不必等到程式執行，目前Kid已經停止開發，由Genshi專案接手。

支援Ajax

對於Ajax技術已經有了很好的支援，Json形式的資料可以被輕鬆地產生

Widget系統

它特有的Widget系統，將常見的網頁元件製造成Widget，可以輕易地重覆使用，當然也可以客制化，例如Ajax的自動完成文字欄，除此之外也有很多第三方的Widget可以使用

活躍的社群

有活躍的社群讓它持續地進步，現在已經在進行2.0版本的開發，向另一個社群PyLons靠攏，他們表示兩者有很多共同的地方，整合更好的想法，互相助開發，會讓社群更加活躍

極快速地開發

有各種理由，讓開發的速度極快，從20分鐘寫Wiki的影片就不難看出開發的生產力有多大的提升

可架構在網頁伺服器上

它可以單獨運作，也可以和各種伺服器結合應用，像常見的Apache，最簡單的方法是使用rewrite，但是它也有提供WSGI來和伺服器結合，所以只要支援WSGI或rewrite都可以整合
